# Urban Route Optimization in Bursa: TSP-with-Neighborhoods

This project presents a comparative performance analysis of different algorithms for solving the **Traveling Salesman Problem (TSP)** using real road network data from the Osmangazi district of Bursa, Turkey. The study evaluates **Greedy Insertion**, **Google OR-Tools**, and **Genetic Algorithm** approaches in terms of route length and computational efficiency.
